# hng-ticket-system-vue

A Vue 3 + Vite port of the TrackiT ticket management demo.

Quick start

1. Install dependencies

```bash
cd hng-ticket-system-vue
npm install
```

2. Run dev

```bash
npm run dev
```

Notes

- This scaffolding mirrors the React app structure: header, footer, home, dashboard, auth routes.
- Pinia is included for state management; implement auth and ticket services similar to the React version.
- Tailwind is configured in `tailwind.config.cjs` and `postcss.config.cjs`.
